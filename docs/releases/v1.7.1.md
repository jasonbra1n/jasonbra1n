# Release v1.7.1: Admin Dashboard, CMS Foundation & AI Integration

This major release establishes the core infrastructure for a dynamic, self-hosted Content Management System (CMS). It introduces a secure Admin Dashboard, database-driven settings, and the foundational service layer for AI integration using Google Gemini.
