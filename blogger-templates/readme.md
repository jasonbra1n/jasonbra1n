# Blogger Templates

This directory contains the XML templates for the jasonbrain.com blogs, which are hosted on the Blogger platform.

- **Music Blog**: [blog.jasonbrain.com](https://blog.jasonbrain.com/)
- **Tech Blog**: [devblog.jasonbrain.com](https://devblog.jasonbrain.com/)

## Purpose

The templates are stored here to be version-controlled and kept in sync with the main website's branding and `STYLE_GUIDE.md`.

When updating the main site's CSS or layout, review these templates to see if corresponding changes are needed to maintain a consistent user experience across all properties.
